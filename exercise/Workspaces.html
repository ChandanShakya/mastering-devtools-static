Exercise 2 - Workspaces

Instructions

  1. Kill the current Node process running this app.
  2. Open the DevTools
  3. Click on the Sources panel
  4. Open Windows Explorer / Finder and navigate to this project
  5. Grab the workspace folder and drag it into the workspace section.
  6. Click allow at the top of your browser window.
  7. You should now see little green circles next to all of the filenames in the sidebar.
  8. Then complete the three tasks outlined in the Workspace!

Finished?

When you're done here, start up the node server again and head to the next lesson
